# Digital Queue Management System

A MERN Stack web application that helps organizations manage customer queues digitally. Users can generate tokens, track their queue status, and administrators can manage organizations, services, and queues through an admin dashboard.

## Features

### User
- Register & Login
- View Organizations
- View Services
- Generate Queue Token
- View Queue History
- Cancel Waiting Token
- Track Token Status

### Admin
- Login as Admin
- Add Organizations
- Add Services
- Edit Services
- Call Next Token
- Complete Token
- View Queue Dashboard
- Delete Queue Records
